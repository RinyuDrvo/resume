## 最も困難だった課題
最初は表ライブラリ`AG Grid`を使用する前提で設計し作成した表に対して、
顧客要望で多数の仕様変更が入り、
当初の想定外のカスタマイズが必要になったことで、
ライブラリで対応できない部分ができたり、使用感が悪くなってしまった。

## 対応内容
ライブラリに頼っていた表の描画を自前のコンポーネントにリプレイスすることで、
顧客要望に柔軟に応えることができた。
イベントや再レンダリングが多数発生する複雑な表をオリジナルで実装することができた。
